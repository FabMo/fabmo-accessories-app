Indexer

What “Run Setup” does

  • Assigns the rotary to its axis and sets the rotary unit values (steps per degree).
  • Sets safe feedrates / speeds for the rotary axis.
  • Installs (or updates) the rotary helper macros.

Setup values for this indexer aren’t loaded yet — this is a scaffold. “Run Setup” is disabled until real config + macros are filled in.

Spindle / VFD

Detects the USB-to-RS485 adapter, identifies the connected VFD by probing known register layouts, and installs the matching profile so spindle RPM control is live. Run this after plugging in a new spindle controller.
