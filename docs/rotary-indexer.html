Rotary Indexer

Offline documentation placeholder. This bundled page is shown when the tool has no internet connection; when online, the app opens the canonical docs on the ShopBot website instead.

Overview

A rotary indexer adds a turning (A) axis to your machine for engraving and machining around cylindrical stock. The Accessories app picks the right axis assignment, unit values (steps per degree), and feedrates for your indexer size, and installs the helper macros.

Setup

  1. Mount the indexer and wire its motor to the rotary axis driver.
  2. In the Accessories app, choose your indexer size and press Run Setup.
  3. Home the machine as usual before running a rotary job.
